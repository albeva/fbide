# Keyword Re-categorisation — Tracking

Source: FB-manual-1.10.1.chm  →  CatPgFunctIndex.html (Functional Keyword List)

Each keyword: `[ ]` = pending, `[x]` = classified.
Right side = assigned target category. Multi-category words use priority
(keyword > type > operator > define > library).

Target categories: `Keywords`, `KeywordTypes`, `KeywordOperators`,
`KeywordConstants`, `KeywordLibrary`, `KeywordPP`, `KeywordAsm1`,
`KeywordAsm2`. Use `-` for non-keywords / shortcuts (e.g. `?`).
